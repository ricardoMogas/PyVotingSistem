Para usar plantillas (templates) en Flask, debes seguir varios pasos, desde la creación de la estructura de carpetas hasta la definición de rutas y el uso de archivos HTML para renderizar contenido dinámico. Flask utiliza Jinja2 como su motor de plantillas por defecto. Pasos para usar plantillas en Flask 1. Instalar Flask Si no lo has hecho aún, instala Flask usando pip: sh pip install Flask 2. Crear la estructura de la aplicación Crea la estructura básica de tu aplicación. Una estructura común es la siguiente: vbnet your_project/ │ ├── app.py ├── templates/ │ ├── base.html │ └── index.html └── static/ └── (optional, for static files like CSS, JS, images) 3. Crear el archivo base de la aplicación En el archivo app.py, configura tu aplicación Flask y define algunas rutas que rendericen plantillas. python from flask import Flask, render_template app = Flask(__name__) @app.route('/') def index(): return render_template('index.html') if __name__ == '__main__': app.run(debug=True) 4. Crear plantillas HTML Crea una plantilla base (base.html) en la carpeta templates. Este archivo servirá como base para otras plantillas y puede incluir bloques que las plantillas específicas podrán sobrescribir. html {% block title %}My Flask App{% endblock %}
