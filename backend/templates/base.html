/check_voto

POST

Verifica si un votante ha votado.

Request Body

{
    "claveElec": "string",
    "id_estado": int,
    "seccion": int
}

Response

{
    "status": boolean,
    "message": "string"
}

/votar

POST

Registra un voto.

Request Body

{
    "seccion": int,
    "id_estado": int,
    "id_candPre": int,
    "claveElec": "string"
}

Response

{
    "status": boolean,
    "message": "string"
}

/votes_per_candidate

GET

Obtiene el número de votos por cada candidato.

Response

{
    "status": true,
    "message": "GET votes per candidate",
    "data": [
        {
            "nombre": "string",
            "total": int
        }
    ]
}

/estados

GET

Obtiene todos los estados.

Response

{
    "status": true,
    "message": "GET votes per candidate",
    "data": [
        {
            "id_estado": int,
            "nombre": "string"
        }
    ]
}

/votes_per_casilla/

GET

Obtiene los votos por casilla.

Response

{
    "status": true,
    "message": "GET votes per casilla",
    "data": [
        {
            "Id_Voto": int,
            "Fecha": "YYYY-MM-DD",
            "Candidato": "string",
            "Id_Casilla": int
        }
    ],
    "Count": [
        {
            "nombre": "string",
            "partido": "string",
            "total": int
        }
    ]
}

/add_resultado

POST

Agrega un resultado.

Request Body

{
    "Id_Casilla": int,
    "PDF": "base64string"
}

Response

{
    "status": boolean,
    "message": "string"
}

/resultados/

GET

Obtiene los resultados de una casilla específica.

Response

{
    "status": true,
    "message": "GET resultados",
    "data": [
        {
            "Id_RE": int,
            "totalVotos": int,
            "Id_Casilla": int,
            "PDF": "base64string",
            "date": "YYYY-MM-DD"
        }
    ]
}

/candidatos

GET

Obtiene todos los candidatos.

Response

{
    "status": true,
    "message": "GET candidatos",
    "data": [
        {
            "Id_CandPre": int,
            "nombreComp": "string",
            "partido": "string",
            "descripcion": "string",
            "imagen": "base64string",
            "imagenPart": "base64string"
        }
    ]
}

/partidos

GET

Obtiene todos los partidos.

Response

{
    "status": true,
    "message": "GET partidos",
    "data": [
        {
            "id_partido": int,
            "nombre": "string",
            "siglas": "string",
            "imagen": "base64string"
        }
    ]
}

/add_partido

POST

Agrega un nuevo partido.

Request Body

{
    "nombre": "string",
    "siglas": "string",
    "imagen": "base64string"
}
